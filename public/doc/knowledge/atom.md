# 插件
## 安装
### apm安装
apm(Atom Package Manager)是atom的包管理工具.
运行安装命令:
```shell
apm install <package-name>
```

# 常见问题
## 自动保存
Atom默认安装了autosave扩展但没有启用,在扩展里面找到它,勾选上enabled就好了.
